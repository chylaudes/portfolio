Boutiq.travel

A traveling blogging app that only shows personal traveling reviews of their facebook friends' excursions. It has a feed that showcases the latest review, and uses Google Map's API to retrieve information about each place the user is reviewing for. Users of the app have the ability to comment their friend's reviews, add a favorite place to their likes page, and post a status for their friends to see. I also added an admin platform for the client to visually see their data allocations.
